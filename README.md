## Primeros pasos

Instalar dependencias

```bash
  npm run prepare
```

Levantar proyecto

```bash
  npm run dev
```

Para instalar una dependencia especifica para Frontend

```bash
  cd ./frontend
  npm i {Opciones de npm} {Nombre de la dependencia}
```
Para instalar una dependencia especifica para Backend

```bash
  cd ./backend
  npm i {Opciones de npm} {Nombre de la dependencia}
```
